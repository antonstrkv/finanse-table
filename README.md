Technologies and libraries:
1. React
2. Redux-toolkit
3. JavaScript
4. Material UI
5. Date-fns
6. Redux-persist
7. Socket io client
8. Eslint
9. Node.js (cors, express, socket io)


Usual features:
1) Application connects to the locally running service
2) Application renders price changes for some tickers in real time
